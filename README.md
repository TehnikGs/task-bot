# Бот-задачник для Telegram 🎙️📋

Начальник **голосом** ставит задачи в общей группе и **голосом** спрашивает статус.
Сотрудник **кнопками** принимает и завершает задачи. Всё видно обоим в одной группе.

Статусы задачи:

| Статус | Что значит |
|---|---|
| 🆕 Не принята | начальник поставил, сотрудник ещё не взял |
| 🔧 В работе | сотрудник принял и выполняет |
| ✅ Завершена | сотрудник завершил |

Голосовые команды начальника (примеры):
- «Поставь задачу убрать цех к вечеру» → создаётся карточка задачи
- «Что сейчас в работе?» → список задач «в работе»
- «Какие задачи завершены?» → список завершённых
- «Что ещё не принято?» → список непринятых
- «Покажи все задачи» → общий список

> Текст работает так же, как голос — можно просто написать боту, удобно для проверки.

## Что нужно сделать один раз (по шагам)

### 1. Создать бота
1. В Telegram найди **@BotFather** → `/newbot` → задай имя и username.
2. Скопируй **токен** (длинная строка) — это `BOT_TOKEN`.
3. Там же: `/setprivacy` → выбери своего бота → **Disable**.
   ⚠️ Это обязательно, иначе бот не услышит голос в группе.

### 2. Получить бесплатный ключ для ИИ
1. Зайди на **https://console.groq.com** (можно через Google-аккаунт).
2. Раздел **API Keys** → **Create API Key** → скопируй ключ — это `LLM_API_KEY`.
   Этот же ключ используется и для распознавания речи. Бесплатно.

### 3. Создать группу
1. Создай в Telegram группу (например, «Задачи»).
2. Добавь туда бота и начальника.
3. (Если бот всё равно не реагирует в группе — сделай его администратором.)

### 4. Заполнить настройки
1. Скопируй `.env.example` в `.env`.
2. Впиши `BOT_TOKEN` и `LLM_API_KEY`.
3. Запусти бота (см. ниже), напиши ему в личку `/id` — он пришлёт твой Telegram ID.
   Так узнай `BOSS_ID` (начальник) и `EMPLOYEE_ID` (сотрудник), впиши в `.env`.

## Запуск на компьютере (Windows, для проверки)

```powershell
cd C:\Users\paref\Documents\task-bot
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
copy .env.example .env   # затем впиши токен и ключ в .env
python bot.py
```

## Запуск на сервере Selectel

```bash
cd /root/myapp
git clone <repo> task-bot      # или git pull, если уже клонирован
cd task-bot
python3 -m venv .venv
.venv/bin/pip install -r requirements.txt
cp .env.example .env           # впиши токен/ключ/ID
cp deploy/task-bot.service /etc/systemd/system/task-bot.service
systemctl daemon-reload
systemctl enable --now task-bot
systemctl status task-bot      # проверить, что Running
journalctl -u task-bot -f      # смотреть логи
```

## Бесплатные варианты распознавания речи

- `STT_BACKEND=groq` (по умолчанию) — облачный Whisper от Groq, бесплатно по тому же ключу.
- `STT_BACKEND=local` — Whisper прямо на сервере, совсем без оплаты запросов.
  Тогда раскомментируй `faster-whisper` в `requirements.txt` и переустанови зависимости.
  Нужен сервер хотя бы с ~2 ГБ свободной памяти; модель задаётся в `STT_LOCAL_MODEL`.

Если ключа ИИ нет вообще — бот всё равно работает, но разбирает команды
по простым ключевым словам (менее «умно»).
